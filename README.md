# Проект 4: Место

**Интро**

Этот проект о чудеснеших городах нашей страны.

**Возможности**
В данном проекте реализованны открытие/закрытие всех поапов и созданием карточек из массива.

**Используемые технологии**

- Адаптивная верстка
- JavaScript
- Работа с массивами
- Валидация форм с помощью JavaScript
